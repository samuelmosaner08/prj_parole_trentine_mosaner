# Dizionario Dialetto Trentino

Un servizio web per consultare il dizionario delle parole ed espressioni del dialetto trentino, con funzionalità di ricerca bilingue, pronuncia audio e materiale fotografico esplicativo.

## Descrizione del Progetto

Il Dizionario Dialetto Trentino è una piattaforma web che permette agli utenti di esplorare e apprendere il dialetto trentino attraverso un'interfaccia intuitiva e accessibile. Il sistema offre traduzioni bidirezionali italiano-dialetto, esempi d'uso contestualizzati e supporto multimediale per una migliore comprensione.

## Funzionalità Principali

- **Ricerca Bilingue**: Cerca parole sia in italiano che in dialetto trentino
- **Pronuncia Audio**: Ascolta la corretta pronuncia delle parole in entrambe le lingue
- **Espressioni Dialettali**: Consulta modi di dire e espressioni tipiche
- **Esempi d'Uso**: Visualizza frasi di esempio per comprendere il contesto
- **Galleria Fotografica**: Immagini esplicative per il riconoscimento di oggetti e concetti
- **Cronologia Ricerche**: Accedi rapidamente alle tue ricerche precedenti
- **Filtri Avanzati**: Filtra per lettera, ordine alfabetico, genere, dizionario e altro

## Requisiti di Sistema

### Prerequisiti
- Database CSV contenente il dizionario completo
- Dominio web per l'hosting
- Permessi per l'utilizzo delle fotografie del Museo Etnografico Trentino San Michele
- Spazio di archiviazione adeguato per contenuti multimediali

### Tecnologie Utilizzate
- Database basato su file CSV
- Gemini Nano Banana Pro per generazione immagini senza copyright
- Design responsive per desktop e mobile
